

# 律师事务所官网 — 实施计划

## 设计风格
- **主色调**：深海蓝 `#0f172a`，强调色：琥珀金 `#d4af37`
- **字体**：Playfair Display（标题）+ Inter（正文），通过 Google Fonts 引入
- **整体风格**：现代、专业、高端感，单页平滑滚动

---

## 模块规划

### 1. 固定导航栏
- 左侧 Logo 文字（事务所名称）
- 中部菜单：首页 / 服务 / 团队 / 案例 / 联系，点击平滑滚动到对应区域
- 右侧 CTA 按钮"预约咨询"，金色边框样式
- 滚动后背景变为玻璃态（backdrop-blur + 半透明背景）
- 移动端：汉堡菜单，侧边抽屉导航

### 2. Hero 横幅区
- 全屏高度，深色背景搭配半透明遮罩
- 大标题"您的权益，我们的使命"（Playfair Display）
- 副标题简介文字
- 双按钮："了解我们"（透明边框）/"立即咨询"（金色填充）
- 入场淡入动画

### 3. 服务领域（6 张卡片）
- 网格布局（桌面 3 列，平板 2 列，手机 1 列）
- 每张卡片：Lucide 图标 + 服务名称 + 简要描述
- 悬停效果：卡片上浮阴影 + 金色边框显现
- 滚动进入视口时逐个淡入

### 4. 律师团队展示
- 网格布局展示 4-6 位律师
- 圆形头像占位图 + 姓名 + 职位
- 姓名下方金色细线装饰
- 滚动触发入场动画

### 5. 信任背书统计
- 深色背景区域，4 个关键数据（如：执业年数、服务客户数、胜诉率、团队人数）
- 数字动画计数器，滚动到视口时触发从 0 增长到目标值
- 金色数字 + 白色描述文字

### 6. 咨询表单
- 字段：姓名、电话、邮箱、案件类型（下拉选择）、案情描述（文本域）
- 表单验证（zod）
- 提交按钮全宽金色
- 提交后显示成功提示（toast）

### 7. 页脚
- 深色背景，三栏布局：
  - 关于我们：事务所简介
  - 快速链接：各区域锚点
  - 联系方式：地址、电话、邮箱
- 底部社交媒体图标行
- 版权信息

---

## 交互与动画
- 使用 Framer Motion 实现滚动触发的 reveal 动画（淡入 + 上移）
- 导航栏滚动监听：固定定位 + 玻璃态过渡
- 统计数字计数器动画
- 卡片悬停上浮效果
- 全站平滑滚动（CSS `scroll-behavior: smooth`）

---

## 技术实现
- 安装 `framer-motion` 依赖
- 自定义 Tailwind 颜色变量（深海蓝、琥珀金）
- 引入 Google Fonts（Playfair Display + Inter）
- 所有内容集中在单页 Index 中，按区域组件化拆分
- 完全响应式适配（桌面、平板、手机）
